# Zoom

**Entity:** Webinar / meetings vendor used for **registration**, **attendance**, and **duration** signals in the sales tracking and buyer-journey plans.

## Role in this program

- **Phase 1:** Manual export or API import feeds “showed” and minutes-attended style metrics; join to leads typically via **normalized email** (decision recorded in [[Product-Phase-Roadmap]]).
- **Later:** Prefer **Reports API** for participant join/leave/duration per current Zoom developer documentation and account scopes.
